# 🎮 Game Save States Archive

A curated collection of **save states for classic and modern video games**, across **multiple platforms and emulators**, ready to use for testing, speedrunning, nostalgia, or game exploration.

> 🚀 100% open and organized by console, emulator, and game — no ROMs or copyrighted content included.

---

## 📌 About This Project

**Game Save States Archive** is an open-source initiative to preserve and share save state files (`.sav`, `.state`, `.save`, etc.) for various video game systems and PC games.

Whether you're a speedrunner, game tester, modder, or just want to revisit specific game moments, this archive offers quick access to **pre-saved points**, **boss fights**, **secret levels**, and more.

---

## 🗂 Folder Structure

```bash
GameSavestates-Archive/
├── PC/
│   └── Game Name/
├── Consoles/
│   ├── NES/
│   │   └── EmulatorName/Game Name/
│   ├── SNES/
│   ├── PS1/
│   ├── PS2/
│   ├── GBA/
│   ├── DS/
│   └── ...more
├── Tools/
│   └── Recommended emulators and configs
Each game folder includes:

    Save state files

    Emulator info

    Optional screenshots

    Game progress notes
```

## ⚙️ How to Use

    Clone or download the repo.

    Navigate to your desired console and game.

    Place the save file in your emulator’s save state folder.

    Make sure you use the same emulator and version indicated in the folder.
